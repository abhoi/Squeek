Squeek

Squeek is a RESTful iOS application that utilizes Twitter's REST APIs to provide a unique interface for the user. Squeek was developed on Apple's XCode Application Development suite. Squeek also uses Twitter's Fabric Framework to authenticate the application id and key.

Brief Implementation Details

  • NSDictionary based modal was created to handle appropriate incoming JSON data
  • JSON data was parsed using appropriate data type methods
  • Tweet ViewControllers and supporting files were created
  • Tweets are displayed according to their length and whether or not they contain a photo
  • User profile page is updated with appropriate user details (Also retrieved from Twitter)
  • Pull to refresh tweets feature with infinite scrolling
